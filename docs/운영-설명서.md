## 1. 관리자 접속 방법

1. 사이트에서 **일반 계정으로 로그인**합니다
2. 주소창에 `enometa-shopingmall.vercel.app/admin` 입력
3. 관리자 대시보드가 나타납니다

> 관리자 권한이 없으면 로그인 페이지로 돌아갑니다.
> 관리자 권한 설정은 아래 "초기 설정" 섹션을 참고하세요.

## 2. 관리자 기능

### 대시보드 (`/admin/dashboard`)
- 오늘 매출, 총 주문 수, 등록 상품 수, 문의 건수 확인

### 상품 관리 (`/admin/products`)
- **상품 목록**: 등록된 전체 상품 확인 + 삭제
- **상품 등록** (`/admin/products/new`):
  - 상품명, 가격, 설명, 카테고리 입력
  - **사이즈**: 버튼 클릭으로 선택 (S, M, L, XL)
  - **컬러**: 컬러 피커로 색상 선택 + 이름 입력, 여러 개 추가 가능
  - **재고**: 선택한 사이즈별 수량 입력
  - **이미지**: 파일을 드래그앤드롭하거나 클릭해서 업로드 (자동으로 URL 생성)
  - URL 직접 입력도 가능
- **상품 수정** (`/admin/products/[id]/edit`): 등록과 동일한 화면에서 수정

### 주문 관리 (`/admin/orders`)
- 전체 주문 목록 확인
- 주문 상태 변경: 결제완료 → 준비중 → 배송중 → 배송완료

### 문의 관리 (`/admin/inquiries`)
- 고객 문의 목록 확인
- 답변 작성

## 3. 쿠폰 시스템

### 자동 발급
- 신규 회원가입 시 **10% 할인쿠폰**이 자동으로 발급됩니다
- 유효기간: 가입일로부터 **30일**
- 최소 주문금액: **50,000원** 이상

### 사이트 안내
- 상단 프로모션 바에 "신규가입 시 10% 할인쿠폰 즉시 지급" 문구가 표시됩니다
- 사용자가 ✕ 버튼으로 닫을 수 있습니다

## 5. 초기 설정 (한 번만)

### 관리자 계정 설정
Supabase Dashboard → **SQL Editor** → 새 쿼리:

```sql
UPDATE auth.users 
SET raw_user_meta_data = jsonb_set(
  COALESCE(raw_user_meta_data, '{}'::jsonb), 
  '{role}', '"admin"'
)
WHERE email = '관리자이메일@입력';
```

### 쿠폰 테이블 (이미 완료)
`docs/coupon-setup.sql` 파일을 SQL Editor에서 실행

### Storage 버킷 (이미 완료)
`product-images` 버킷 생성 (Public)

### Vercel 환경변수
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `NEXT_PUBLIC_TOSS_CLIENT_KEY`
- `TOSS_SECRET_KEY`
