## 2. 와이어프레임 (ASCII)

### 2.1 인트로 페이지 (/) — 퀄리티 목표 95%

```
┌─────────────────────────────────────────────┐
│                                             │
│                                             │
│                                             │
│            E n o m e t a _                  │  ← 타이핑 애니메이션 (1줄씩 순차)
│            Shopping Mall                     │
│            Portfolio                         │
│                                             │
│     ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐         │  ← 5장 이미지 카드 stagger 등장
│     │ 1 │ │ 2 │ │ 3 │ │ 4 │ │ 5 │         │    (0.15s 간격)
│     └───┘ └───┘ └───┘ └───┘ └───┘         │
│                                             │
│            [ ENTER SHOP ]                   │  ← 페이지 트랜지션으로 /shop 이동
│                                             │
└─────────────────────────────────────────────┘
```

**인터랙션 시퀀스:**
1. 화면 진입 (0s) → 배경 페이드인
2. "Enometa" 타이핑 시작 (0.3s) → 0.8s 완료
3. "Shopping Mall" 타이핑 (1.0s) → 1.5s 완료
4. "Portfolio" 타이핑 (1.7s) → 2.0s 완료
5. 이미지 카드 5장 stagger 등장 (2.2s ~ 2.9s)
6. "ENTER SHOP" 버튼 페이드인 (3.0s)
7. 전체 시퀀스 3초 이내 완료

**모바일:**
- 이미지 카드 3열→2열로 축소
- 타이핑 폰트 크기 축소
- 카드와 버튼 간격 조정

## 3. 인터랙션 명세

### 3.1 페이지 트랜지션 (FR-50)

```
현재 페이지                      다음 페이지
┌──────────┐                  ┌──────────┐
│          │  → opacity: 0    │          │
│          │    + y: -20px    │          │  ← opacity: 0→1
│          │    (0.2s)        │          │     + y: 20px→0
│          │                  │          │     (0.3s)
└──────────┘                  └──────────┘

기술: Framer Motion AnimatePresence
  exit: { opacity: 0, y: -20 }
  initial: { opacity: 0, y: 20 }
  animate: { opacity: 1, y: 0 }
  transition: { duration: 0.3, ease: "easeOut" }
```

### 3.2 호버 이미지 스왑 (FR-13)

```
기본 상태              호버 상태
┌──────────┐         ┌──────────┐
│ 에디토리얼 │ ──────→ │   누끼    │
│  (모델컷) │         │ (플랫컷)  │
│ opacity:1 │         │ opacity:1 │
└──────────┘         └──────────┘

구현: .hover-img { opacity: 0; position: absolute; }
      :hover .hover-img { opacity: 1; }
      transition: opacity 0.4s cubic-bezier(0.165, 0.84, 0.44, 1)
```

## 4. 반응형 전략

| 요소 | Desktop (1025px+) | Mobile (≤1024px) |
|------|-------------------|------------------|
| 헤더 | 호버 드롭다운 메가메뉴 | 햄버거 → 사이드메뉴 |
| 상품 그리드 | 2/3/4열 전환 | 2열 고정 |
| 상세 이미지 | 좌측 60% 세로 스크롤 | 가로 스와이프 갤러리 |
| 상세 정보 | 우측 40% sticky | 이미지 아래 배치 |
| 장바구니 | 사이드 드로어 400px | 사이드 드로어 100% |
| 결제 | 2컬럼 (폼+요약) | 1컬럼 (세로 나열) |
| 관리자 | 사이드바 + 본문 | 사이드바 접힘 (햄버거) |
| 카드 호버 | 이미지 스왑 + 미리보기 | 없음 (터치 디바이스) |
| 좌우 패딩 | 20~50px | 20~40px |
